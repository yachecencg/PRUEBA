Introducción a HTML

HTML (HyperText Markup Language) es el lenguaje estándar para crear páginas web.

Es un lenguaje de marcado, lo que significa que utiliza etiquetas para definir la estructura y el contenido de una página.

Conceptos Clave

Etiquetas y Atributos

Las etiquetas se encierran entre corchetes angulares, por ejemplo: <p> para un párrafo.

Muchos elementos tienen atributos que proporcionan información adicional. Por ejemplo, en un enlace:

Visita MDN Web Docs sobre HTML

(El atributo target="_blank" abre el enlace en una nueva pestaña)

Estructura del Documento

  • <!DOCTYPE html>: Declara el tipo de documento.
  • <html>: La raíz de todo el documento HTML.
  • <head>: Contiene metadatos (información sobre la página, no visible directamente).
  • <body>: Contiene todo el contenido visible de la página.

Elementos Semánticos

Ayudan a organizar el contenido de manera significativa:

  • <header>: Contenido introductorio o un grupo de enlaces de navegación.
  • <nav>: Sección de enlaces de navegación.
  • <main>: Contenido principal único del documento.
  • <section>: Agrupa contenido relacionado.
  • <article>: Contenido independiente y auto-contenido.
  • <footer>: Pie de página de un documento o sección.
